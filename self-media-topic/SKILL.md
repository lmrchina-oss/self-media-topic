# 自媒体选题

## 任务目标
- 本 Skill 用于：追踪指定领域和平台的热点话题，多维度评估选题潜力，提供针对性的标题优化建议
- 能力包含：热点搜索、选题评分、平台适配、标题优化、执行建议
- 触发条件：用户需要选择内容方向、分析热点趋势、优化平台标题或制定内容策略

## 操作步骤

### 1. 解析用户输入
从用户请求中提取以下信息：
- **领域关键词**：目标领域（如 AI工具、美妆、健身等）
- **目标平台**：分析哪些平台（小红书、B站、公众号）
- **时间范围**：热点时间范围（今天、近一周、近一个月、近三个月）
- **选题数量**：推荐多少个选题（默认5个）

### 2. 搜索热点数据
使用内置搜索功能获取相关热点：
- 构建搜索查询："{关键词} {时间范围} 热点 新闻 趋势 {平台}"
- 分析搜索结果，提取潜在选题
- 记录每个选题的热度、讨论度、时间信息

### 3. 多维度评分
基于 [references/scoring_criteria.md](references/scoring_criteria.md) 对每个选题进行评分：

**6个维度（总分100分）：**
1. **热度趋势**（25分）：热榜表现、搜索趋势、讨论热度
2. **时间匹配度**（20分）：时间敏感性、时效性价值
3. **平台适配度**（20分）：根据目标平台的用户画像和内容偏好评估
4. **内容稀缺性**（15分）：竞争度分析、差异化空间
5. **可执行性**（10分）：制作难度、资源需求
6. **商业价值**（10分）：变现潜力、长期价值

**评分要点：**
- 真实优先：基于实际数据评分，不夸大
- 平台差异化：不同平台的评分标准有所侧重
- 实用性至上：确保推荐的是用户能实际执行的选题
- 避免同质化：优先推荐有差异化空间的选题

### 4. 筛选和排序
- 按总分从高到低排序
- 选出Top {选题数量} 个选题
- 确保每个选题在各个维度都有明确得分理由

### 5. 优化平台标题
基于 [references/platform_guidelines.md](references/platform_guidelines.md) 为每个选题优化标题：

**小红书标题公式：** 疑问式 + 利益点 + emoji + 关键词
- 字数：20-30字
- 特征：实用性强、情感共鸣、信息密度高

**B站标题公式：** 悬念/反差 + 数据/数字 + 垂直领域 + 情绪词
- 字数：25-40字
- 特征：内容深度、创意独特、互动性强

**公众号标题公式：** 数字 + 痛点 + 解决方案 + 紧迫感
- 字数：20-35字
- 特征：观点犀利、数据支撑、情绪共鸣

为每个平台提供3个标题建议，并说明优化理由。

### 6. 给出执行建议
为每个选题提供具体可执行的创作建议：
- **发布时间**：建议的时间段和原因
- **内容形式**：图文/视频/长文等
- **关键元素**：必须包含的关键元素
- **注意事项**：制作过程中需要注意的点

### 7. 格式化输出
严格按照 [references/output_template.md](references/output_template.md) 的格式输出结果。

## 资源索引

### 核心评分标准（6维度总分100分）

**1. 热度趋势（25分）**
- 热榜表现（10分）：是否进入过热榜、排名如何
  - 9-10分：进入多个平台热榜前10
  - 7-8分：进入平台热榜前50
  - 5-6分：小范围热度，未上热榜
- 搜索趋势（8分）：相关关键词搜索量变化
  - 7-8分：搜索量大幅上升（>50%）
  - 5-6分：稳步上升（20-50%）
  - 3-4分：略有上升（<20%）
- 讨论热度（7分）：社交媒体讨论度
  - 6-7分：讨论度极高，话题参与度强
  - 4-5分：讨论度较高
  - 2-3分：讨论度一般

**2. 时间匹配度（20分）**
- 时间敏感性（10分）：在选定时间内的表现
  - 9-10分：快速爆发，时效性极强
  - 7-8分：有明显上升趋势
  - 5-6分：趋势平稳
- 时效性价值（10分）：信息新鲜度
  - 9-10分：非常新，信息价值高
  - 7-8分：较新，有一定价值
  - 5-6分：一般，价值有限

**3. 平台适配度（20分）**

**小红书**（20分满分）：
- 用户画像：18-35岁女性为主，注重实用性和情感价值
- 内容偏好：干货分享、生活感悟、产品测评、种草推荐
- 爆款特征：实用性强、情感共鸣、审美在线、信息密度高
- 评分标准：17-20分高度契合，13-16分比较契合

**B站**（20分满分）：
- 用户画像：18-35岁男性略多，注重深度和创意
- 内容偏好：知识科普、技术教程、创意视频、游戏娱乐
- 爆款特征：内容深度、创意独特、互动性强、专业度高
- 评分标准：17-20分高度契合，13-16分比较契合

**公众号**（20分满分）：
- 用户画像：25-45岁职场人为主，注重观点和价值
- 内容偏好：行业分析、职场感悟、商业洞察、情绪价值
- 爆款特征：观点犀利、数据支撑、情绪共鸣、实用干货
- 评分标准：17-20分高度契合，13-16分比较契合

**4. 内容稀缺性（15分）**
- 竞争度分析（8分）：同类内容数量
  - 7-8分：同类内容极少，市场空白
  - 5-6分：同类内容不多
  - 3-4分：同类内容较多
- 差异化空间（7分）：独特切入角度
  - 6-7分：有明显差异化角度
  - 4-5分：有一定差异化空间
  - 2-3分：差异化空间有限

**5. 可执行性（10分）**
- 制作难度（5分）：制作复杂度
  - 5分：制作简单，无需特殊技能
  - 3-4分：制作中等，需要一定技能
  - 1-2分：制作困难，需要专业技能
- 资源需求（5分）：所需资源
  - 5分：无需特殊资源
  - 3-4分：需要少量资源
  - 1-2分：需要较多资源

**6. 商业价值（10分）**
- 变现潜力（5分）：广告和带货潜力
  - 5分：广告价值高，带货潜力强
  - 3-4分：有一定变现潜力
  - 1-2分：变现潜力有限
- 长期价值（5分）：持续产出潜力
  - 5分：可持续产出，建立品牌
  - 3-4分：有一定持续产出空间
  - 1-2分：一次性内容

### 平台标题优化策略

**小红书公式：疑问式 + 利益点 + emoji + 关键词**
- 字数：20-30字
- 示例："每天5分钟，这个方法让我省下3000元 💰"
- 要点：开头制造悬念，中间给出利益点，加入emoji，避免标题党

**B站公式：悬念/反差 + 数据/数字 + 垂直领域 + 情绪词**
- 字数：25-40字
- 示例："耗时300小时，我终于做成了XX，结果让人意外！"
- 要点：开头制造悬念，用数字增加可信度，明确领域，加入情绪词

**公众号公式：数字 + 痛点 + 解决方案 + 紧迫感**
- 字数：20-35字
- 示例："2024年最值得关注的5个AI工具，第3个我每天都在用"
- 要点：用数字增加吸引力，直击痛点，给出方案，营造紧迫感

## 使用示例

**用户请求：**
"帮我分析AI工具领域的小红书和公众号热点，近一周时间，推荐5个选题"

**执行流程：**
1. 搜索："AI工具 近一周 热点 新闻 趋势 小红书 公众号"
2. 识别潜在选题：如"AI写作工具对比"、"AI设计工具测评"等
3. 基于6维度评分，筛选Top 5
4. 为小红书和公众号各优化3个标题
5. 给出每个选题的执行建议
6. 按标准格式输出结果

## 注意事项
- 仅在需要时详细读取参考文档，保持上下文简洁
- 评分时要基于真实数据，避免主观臆断
- 标题优化要符合各平台特征，避免模板化
- 执行建议要具体可操作，避免空泛建议
- 充分利用智能体的语言理解和生成能力，专注于策略和创意
