## 字段填写说明

### 选题 #{序号}：{选题名称}
- **序号**：从1开始递增
- **选题名称**：简洁明了地描述选题核心内容

### 爆款评分
- **总分**：6个维度分数的总和，范围0-100

### 详细评分

每个维度的格式：**维度名：分数/满分 | 评分理由**

**热度趋势（25分）**
- 分数范围：0-25
- 评分理由说明：热榜表现、搜索趋势、讨论热度的具体情况

**时间匹配度（20分）**
- 分数范围：0-20
- 评分理由说明：时间敏感性和时效性价值的分析

**平台适配度（20分）**
- 分数范围：0-20
- 评分理由说明：与目标平台用户画像和内容偏好的匹配度

**内容稀缺性（15分）**
- 分数范围：0-15
- 评分理由说明：竞争度和差异化空间的分析

**可执行性（10分）**
- 分数范围：0-10
- 评分理由说明：制作难度和资源需求的评估

**商业价值（10分）**
- 分数范围：0-10
- 评分理由说明：变现潜力和长期价值的分析

### 选题分析

详细分析应该包含：
1. **热点背景**：这个选题的热点来源、数据支撑
2. **价值分析**：为什么这个选题有价值，解决了什么问题
3. **时机分析**：为什么现在适合做这个选题
4. **差异化**：这个选题的独特之处

示例：
```
随着AI技术的快速发展，AI写作工具成为近期热点话题。搜索数据显示，"AI写作工具"相关关键词搜索量在一周内增长200%，多个平台的讨论热度持续上升。这个选题的价值在于帮助创作者提高效率，解决内容创作痛点。当前正值AI工具普及期，用户需求旺盛，且市场上关于AI写作工具的深度测评还不多，有较大的差异化空间。
```

### 目标平台标题建议

每个平台提供3个标题建议，每个标题格式：
- **标题内容** | **优化理由**

**优化理由**应该说明：
- 使用了哪些标题优化技巧
- 为什么这样优化
- 预期达到什么效果

示例：
```
小红书：
- "每天5分钟，这个AI写作工具让我效率提升10倍 ✨" | 使用疑问式开头（每天5分钟），给出具体利益点（效率提升10倍），加入emoji（✨）增加亲和力
- "终于找到！最适合新手的AI写作工具，亲测有效 😍" | 制造悬念（终于找到），强调低门槛（适合新手、亲测有效），使用emoji（😍）
- "后悔没早点知道！这个AI工具太好用了 💡" | 痛点共鸣（后悔没早点知道），强烈推荐（太好用了），使用emoji（💡）
```

### 执行建议

**发布时间**
- 建议的时间段（如：工作日晚上8-10点）
- 选择该时间段的原因（如：用户活跃度高）

**内容形式**
- 建议的内容形式（如图文、短视频、长文等）
- 选择该形式的理由

**关键元素**
- 必须包含的核心内容（如：工具功能介绍、使用场景、效果对比等）
- 必须包含的视觉元素（如：截图、演示视频、数据图表等）

**注意事项**
- 制作过程中的重要提醒（如：保持客观、数据准确、避免过度宣传等）
- 可能遇到的问题及解决方案

示例：
```
执行建议：
- 发布时间：工作日晚上8-10点，此时用户活跃度最高，且有时间阅读深度内容
- 内容形式：图文+短视频，图文详细说明，短视频展示实际操作
- 关键元素：工具功能介绍、实际使用场景演示、效果对比数据、个人使用体验
- 注意事项：保持客观中立，避免过度宣传；数据要准确，可提供验证方式；互动引导要自然
```

## 注意事项

1. **格式一致性**：所有选题必须使用相同的格式
2. **评分准确性**：基于实际数据评分，避免主观臆断
3. **标题质量**：标题要符合各平台特征，避免模板化
4. **建议具体**：执行建议要具体可操作，避免空泛
5. **逻辑清晰**：分析要逻辑严密，有理有据
6. **语言简洁**：表达简洁明了，避免冗长
