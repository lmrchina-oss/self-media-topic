## 热度趋势（25分）

### 热榜表现（10分）
- **9-10分**：进入多个平台热榜前10，或单一平台热榜前3
- **7-8分**：进入平台热榜前50，或有一定知名度
- **5-6分**：在小范围有热度，但未上热榜
- **0-4分**：无明显热榜表现

**示例：**
- 某话题在小红书热榜第2位 → 10分
- 某话题在公众号阅读量10万+ → 8分
- 某话题在小圈子讨论活跃 → 5分

### 搜索趋势（8分）
- **7-8分**：相关关键词搜索量近期大幅上升（>50%）
- **5-6分**：搜索量稳步上升（20-50%）
- **3-4分**：搜索量略有上升（<20%）
- **0-2分**：搜索量无变化或下降

**示例：**
- "AI写作工具"搜索量一周内增长200% → 8分
- "健康饮食"搜索量稳定增长30% → 5分
- "传统美食"搜索量持平 → 1分

### 讨论热度（7分）
- **6-7分**：社交媒体讨论度极高，话题参与度强
- **4-5分**：讨论度较高，有一定话题参与度
- **2-3分**：讨论度一般，参与度有限
- **0-1分**：讨论度低，几乎无参与

**示例：**
- 微博话题阅读量过亿 → 7分
- 知乎相关回答500+ → 5分
- 仅有少量评论 → 2分

## 时间匹配度（20分）

### 时间敏感性（10分）
- **9-10分**：在选定时间内快速爆发，时效性极强
- **7-8分**：在选定时间内有明显上升趋势
- **5-6分**：趋势平稳，无明显时间敏感性
- **0-4分**：趋势下降或过时

**示例：**
- 新技术发布当天 → 10分
- 近一周持续上升 → 8分
- 长期话题无波动 → 5分

### 时效性价值（10分）
- **9-10分**：非常新，信息价值高
- **7-8分**：较新，有一定信息价值
- **5-6分**：一般，信息价值有限
- **0-4分**：过时，信息价值低

**示例：**
- 最新发布的AI工具 → 10分
- 近期热门的生活方式 → 7分
- 经典话题 → 3分

## 平台适配度（20分）

### 小红书特征（20分满分）
**用户画像**：18-35岁女性为主，注重实用性和情感价值
**内容偏好**：干货分享、生活感悟、产品测评、种草推荐
**爆款特征**：实用性强、情感共鸣、审美在线、信息密度高

**评分标准：**
- **17-20分**：高度契合，符合用户画像和内容偏好
- **13-16分**：比较契合，有一定适配度
- **9-12分**：一般适配，需要调整方向
- **0-8分**：不契合，不适合该平台

### B站特征（20分满分）
**用户画像**：18-35岁，男性略多，注重深度和创意
**内容偏好**：知识科普、技术教程、创意视频、游戏娱乐
**爆款特征**：内容深度、创意独特、互动性强、专业度高

**评分标准**：
- **17-20分**：高度契合，符合用户画像和内容偏好
- **13-16分**：比较契合，有一定适配度
- **9-12分**：一般适配，需要调整方向
- **0-8分**：不契合，不适合该平台

### 公众号特征（20分满分）
**用户画像**：25-45岁职场人为主，注重观点和价值
**内容偏好**：行业分析、职场感悟、商业洞察、情绪价值
**爆款特征**：观点犀利、数据支撑、情绪共鸣、实用干货

**评分标准：**
- **17-20分**：高度契合，符合用户画像和内容偏好
- **13-16分**：比较契合，有一定适配度
- **9-12分**：一般适配，需要调整方向
- **0-8分**：不契合，不适合该平台

**多平台评分规则：**
- 如果用户选择多个平台，取各平台分数的平均值
- 如果某个平台完全不适配，需在分析中说明并建议排除

## 内容稀缺性（15分）

### 竞争度分析（8分）
- **7-8分**：同类内容极少，市场空白
- **5-6分**：同类内容不多，有一定空间
- **3-4分**：同类内容较多，竞争激烈
- **0-2分**：同类内容极多，竞争白热化

**示例：**
- 全新领域话题 → 8分
- 热门话题但角度独特 → 5分
- 饱和市场话题 → 2分

### 差异化空间（7分）
- **6-7分**：有明显的差异化角度和切入空间
- **4-5分**：有一定的差异化空间
- **2-3分**：差异化空间有限
- **0-1分**：几乎没有差异化空间

**示例：**
- 新技术应用场景 → 7分
- 结合个人经验的分享 → 5分
- 常规内容 → 2分

---

## 可执行性（10分）

### 制作难度（5分）
- **5分**：制作简单，无需特殊技能
- **3-4分**：制作中等，需要一定技能
- **1-2分**：制作困难，需要专业技能
- **0分**：制作极其困难，不建议

**示例：**
- 干货总结类 → 5分
- 深度分析类 → 3分
- 专业测评类 → 1分

### 资源需求（5分）
- **5分**：无需特殊资源，立即可执行
- **3-4分**：需要少量资源（如工具、资料）
- **1-2分**：需要较多资源（如设备、数据）
- **0分**：需要大量资源，难以执行

**示例：**
- 经验分享 → 5分
- 工具测评 → 3分
- 产品评测（需要样品） → 1分

---

## 商业价值（10分）

### 变现潜力（5分）
- **5分**：广告价值高，带货潜力强
- **3-4分**：有一定变现潜力
- **1-2分**：变现潜力有限
- **0分**：无明显变现潜力

**示例：**
- 产品测评类 → 5分
- 干货分享类 → 3分
- 纯观点类 → 1分

### 长期价值（5分）
- **5分**：具备持续产出潜力，可建立品牌
- **3-4分**：有一定持续产出空间
- **1-2分**：一次性内容，长期价值低
- **0分**：无长期价值

**示例：**
- 系列教程 → 5分
- 行业观察 → 3分
- 偶发热点 → 1分

## 总分计算方式

**总分 = 热度趋势(25) + 时间匹配度(20) + 平台适配度(20) + 内容稀缺性(15) + 可执行性(10) + 商业价值(10) = 100分**

**评分等级：**
- **90-100分**：爆款潜力极高，强烈推荐
- **80-89分**：爆款潜力较高，优先推荐
- **70-79分**：有潜力，可以考虑
- **60-69分**：潜力一般，谨慎选择
- **60分以下**：不推荐

**注意事项：**
- 每个维度都要给出具体的评分理由
- 评分要基于实际数据，避免主观臆断
- 如果某个维度数据不足，说明原因并给出合理估算
